# FreeCell
Projeto da disciplina "Programação Modular"

Instruções de execução
---------------------

1. Abrir CMD
2. Ir até o diretório Batches

3. Para compilar todos módulos:
	3.1. Rodar o CompilaTudo.bat

	Para compilações individuais de cada módulo:
	Rodar:  - Módulo Lista -> CompilaLista.bat
		- Módulo Baralho -> CompilaBaralho.bat
		- Módulo Células Extras -> CompilaCE.bat
		- Módulo Coluna Visível -> CompilaCV.bat
		- Módulo Célula Base -> CompilaCB.bat

4. Para rodar todos os testes:
	4.1. Rodar o TestaTudo.bat

	Para testes individuais de cada módulo:
	Rodar:  - Módulo Lista -> TestaLista.bat
		- Módulo Baralho -> TestaBaralho.bat
		- Módulo Células Extras -> TestaCE.bat
		- Módulo Coluna Visível -> TestaCV.bat
		- Módulo Célula Base -> TestaCB.bat

OBS.: O ArcaboucoTeste.lib já está incluido no zip.
